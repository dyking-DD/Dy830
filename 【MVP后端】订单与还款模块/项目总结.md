# 汽车分期管理平台 - 订单与还款模块开发总结

## 项目概述

已完成汽车分期管理平台的订单管理和还款管理模块后端API开发，包含完整的订单生命周期管理、还款计划管理、抵押登记管理和驾驶舱统计功能。

## 功能模块

### A. 订单管理模块

#### 1. 订单创建
- **接口**: `POST /api/v1/orders`
- **功能**: 创建新订单，自动生成订单ID
- **字段**: 客户信息、车辆信息、贷款信息等

#### 2. 订单列表查询
- **接口**: `GET /api/v1/orders`
- **功能**: 分页查询订单列表
- **筛选**: 支持按阶段、关键词、日期范围筛选
- **返回**: 订单基本信息列表

#### 3. 订单详情查询
- **接口**: `GET /api/v1/orders/{order_id}`
- **功能**: 查询订单完整详情
- **包含**: 客户信息、车辆信息、贷款信息、垫资信息、GPS信息、归档信息、阶段信息

#### 4. 订单阶段更新
- **接口**: `PUT /api/v1/orders/{order_id}/stage`
- **功能**: 更新订单阶段
- **特点**: 严格状态机检查，不允许越级流转
- **状态流转**: 已定义完整的21个阶段流转规则

### B. 还款管理模块

#### 1. 还款计划生成
- **接口**: `POST /api/v1/repayments/plans/generate`
- **功能**: 为订单自动生成多期还款计划
- **计算**: 自动计算每月还款日（起始日 + n个月）
- **生成**: 自动生成每期的plan_id

#### 2. 还款计划查询
- **接口**: `GET /api/v1/repayments/plans`
- **功能**: 查询还款计划列表
- **筛选**: 支持按订单ID、状态筛选
- **关联**: 自动关联客户姓名

#### 3. 还款计划详情
- **接口**: `GET /api/v1/repayments/plans/{plan_id}`
- **功能**: 查询单个还款计划详情
- **包含**: 还款计划信息和客户信息

#### 4. 还款记录录入
- **接口**: `POST /api/v1/repayments/records`
- **功能**: 录入还款记录
- **自动计算**: 是否逾期、逾期天数
- **自动更新**: 所有期数还清后自动更新订单状态为"已结清"

#### 5. 还款记录查询
- **接口**: `GET /api/v1/repayments/records`
- **功能**: 查询还款记录列表
- **筛选**: 支持按订单ID、日期范围筛选

#### 6. 逾期检测
- **接口**: `GET /api/v1/repayments/check-overdue`
- **功能**: 自动检测逾期还款计划
- **处理**: 找出所有应还日期<今天且status=正常的计划，自动更新为"逾期"
- **返回**: 逾期plan_id列表、逾期总金额

#### 7. 还款统计
- **接口**: `GET /api/v1/repayments/stats`
- **统计项**: 
  - 今日待还总额
  - 本月待还总额
  - 本月实际已还总额
  - 逾期笔数、逾期金额
  - 逾期3天内笔数
  - 逾期7天内笔数
  - 正常笔数

### C. 抵押管理模块

#### 1. 抵押登记
- **接口**: `POST /api/v1/mortgage`
- **功能**: 创建抵押登记
- **自动**: 生成mortgage_id

#### 2. 抵押查询
- **接口**: `GET /api/v1/mortgage`
- **功能**: 查询抵押列表
- **筛选**: 支持按订单ID、状态筛选

#### 3. 抵押详情
- **接口**: `GET /api/v1/mortgage/{order_id}`
- **功能**: 根据订单ID查询抵押详情

#### 4. 解除抵押
- **接口**: `POST /api/v1/mortgage/{order_id}/release`
- **功能**: 解除抵押
- **自动**: 更新抵押状态为"已解押"，订单状态为"已完结"

#### 5. 抵押统计
- **接口**: `GET /api/v1/mortgage/stats`
- **统计项**: 抵押总笔数、抵押中笔数、已解押笔数、30天内到期笔数

### D. 驾驶舱模块

#### 全局驾驶舱
- **接口**: `GET /api/v1/dashboard`
- **汇总数据**:
  - 订单统计（今日新增/本月新增/总订单）
  - 垫资统计（垫资余额/今日垫资/逾期数）
  - GPS统计（在线数/离线数/告警数）
  - 归档统计（完成率/待归档数）
  - 还款统计（逾期数/逾期金额/今日待还）
  - 抵押统计（抵押中/30天到期）

## 数据库设计

### orders（订单表）
- 主键：order_id
- 字段：客户信息、车辆信息、贷款信息、阶段信息等
- 特点：包含完整的订单生命周期字段

### repayment_plans（还款计划表）
- 主键：plan_id
- 外键：order_id
- 字段：期数、应还日期、应还金额、实还日期、实还金额、状态、逾期天数

### repayment_records（还款记录表）
- 主键：record_id
- 外键：plan_id, order_id
- 字段：实还金额、实还日期、支付方式、备注

### mortgage（抵押登记表）
- 主键：mortgage_id
- 外键：order_id
- 字段：抵押银行、登记日期、到期日期、登记证编号、状态、解押日期
