# TrendMatrix

A-share 趋势跟踪量化策略，基于大盘择时 + 个股动量 + 移动止损。

## 策略逻辑

```
入场信号：
  - 大盘（沪深300）在 MA10 之上
  - 个股收盘价 > MA10
  - 个股放量 > 1.2倍（对比5日均量）
  → 次日开盘买入

出场信号：
  - 硬止损：-5%
  - MA10 止损：收盘跌破 MA10
  - 分批止盈：+8% 卖出的一半，剩余移动止损

仓位管理：
  - 核心仓（4只）：浪潮信息、北方华创、英维克、中科曙光
  - 观察仓（2只）：中微公司、阳光电源
  - 核心仓 20万/只，观察仓 5万/只

板块方向：
  - 液冷散热（英维克、高澜股份）
  - 算力基建（浪潮信息、中科曙光、中际旭创）
  - 半导体设备（北方华创、中微公司、拓荆科技）
  - 储能（阳光电源、科陆电子）
```
