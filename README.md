# AI CAR — 汽车分期智能管理平台

> 基于多模态AI的汽车分期助贷机构业务管理平台

## 🎯 项目简介

面向汽车分期公司（助贷机构）的数字化业务管理平台，实现从客户进店接单、垫资智能管理、银行审批跟进、GPS实时监控、资料归档到还款全流程的数字化管理。

## 🌟 核心功能

- **垫资管理** — 申请→审批→出账→计息→还款→逾期监控
- **GPS监控** — 设备管理→定位追踪→分级告警→驾驶舱
- **资料归档** — 标准清单→上传→OCR识别→归档状态追踪
- **全链路通知** — 9大节点自动触发钉钉/短信通知
- **还款管理** — 还款计划→自动扣款→逾期预警
- **抵押管理** — 抵押登记→到期提醒→解押

## 🛠️ 技术栈

- **后端：** Python 3 + FastAPI + Pydantic + SQLite
- **前端：** HTML/CSS/JS（原型）→ React（规划）
- **通知：** 钉钉机器人（已接入）
- **GPS：** 预留接口（可接博实结/博云车联）
- **OCR：** 预留接口（可接阿里云OCR）
