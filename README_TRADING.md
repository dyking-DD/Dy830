# 🎯 Mac 模拟交易盘

## 快速启动（两种方式）

### 方式1：双击启动（推荐）
直接双击文件 **`启动交易盘.command`**，自动打开浏览器

### 方式2：终端启动
```bash
cd daily_stock_analysis
python3 trading_desk.py
```

> 启动后会自动打开浏览器，无需手动输入地址

## 功能

- 💰 **实时资产面板** - 总资产、现金、持仓市值、盈亏
- 📈 **资金曲线图** - 30日走势
- 📦 **持仓分布图** - 饼图展示
- ⚡ **快速交易** - 买入/卖出股票
- 📋 **持仓明细** - 成本、现价、盈亏
- 📝 **交易记录** - 最近20笔交易

## 交易规则

- 初始资金: ¥1,000,000
- 最小交易单位: 100股
- 手续费: 0.03% (最低5元)
- 印花税: 卖出时0.1%

## 数据存储

所有数据自动保存到 `trading_account.json`，重启后数据不丢失。

## 文件说明

| 文件 | 用途 |
|------|------|
| `启动交易盘.command` | **双击运行**，一键启动 |
| `trading_desk.py` | 主程序（带自动开浏览器） |
| `trading_account.json` | 账户数据（自动创建） |

## 快捷键

- **Cmd+R** - 刷新数据
- **Cmd+W** - 关闭标签页
- **Ctrl+C** (终端) - 停止服务
