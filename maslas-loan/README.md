# 飞书小程序 - 汽车分期报单系统

## 📱 文件说明

### 1. loan-form.html - 报单表单
业务员提交报单的表单页面

**功能**：
- 客户信息录入（姓名、电话、身份证）
- 业务员信息
- 车辆信息选择
- 金融信息（车价、首付、贷款、期数）
- 自动计算贷款金额
- 表单验证
- 提交到后端API

**字段**：
- 客户姓名（必填）
- 联系电话（必填，11位手机号）
- 身份证号（必填，18位）
- 业务员姓名（必填）
- 车型（必填，下拉选择）
- 车价（必填，万元）
- 首付（必填，万元）
- 贷款金额（自动计算）
- 贷款期数（必填，12-60期）
- 备注说明（可选）

---

### 2. my-orders.html - 我的报单
业务员查看自己的报单列表

**功能**：
- 显示报单统计（总报单、待审批、已通过）
- 报单列表展示
- 状态标识（待审批、已通过、已拒绝）
- 自动刷新（30秒）

**显示信息**：
- 报单编号
- 客户姓名
- 联系电话
- 车型
- 提交时间
- 贷款金额
- 贷款期数
- 审批状态

## 🚀 部署方式

### 方式1：本地部署（推荐测试）

1. 将文件放到Web服务器：
```bash
cd ~/Desktop/2-多节点智能体汽车分期协作系统_软著/
python3 -m http.server 8080
```

2. 访问地址：
- 报单表单：http://localhost:8080/feishu-app/loan-form.html
- 我的报单：http://localhost:8080/feishu-app/my-orders.html

### 方式2：部署到Nginx

1. 复制文件到Nginx目录：
```bash
sudo cp -r feishu-app/* /usr/local/nginx/html/
```

2. 重启Nginx：
```bash
sudo nginx -s reload
```

3. 访问地址：
- 报单表单：http://your-domain.com/loan-form.html
- 我的报单：http://your-domain.com/my-orders.html

### 方式3：部署到云服务器（阿里云/腾讯云）

1. 上传文件到服务器
2. 配置Nginx
3. 绑定域名
4. 配置HTTPS（可选）

## 💡 使用说明

### 业务员操作

1. **提交报单**
   - 打开 `loan-form.html`
   - 填写报单信息
   - 点击"提交报单"
   - 等待审核

2. **查看进度**
   - 打开 `my-orders.html`
   - 查看报单列表
   - 查看审批状态

### 管理员操作

1. **查看所有报单**
   - 打开控制面板
   - 查看实时统计
   - 筛选报单

2. **审批报单**
   - 点击报单详情
   - 查看客户信息
   - 进行审批决策
