# 🚀 汽车分期报单系统 - 快速入门

## 📱 一分钟了解

**这是什么？**
汽车分期贷款的在线报单系统，不用跑公司，手机上就能提交申请！

**核心功能**
- 📝 在线报单（3分钟完成）
- 📋 实时查看审批进度
- 🔔 审批结果自动通知

---

## 🎯 三步上手

### 业务员（提交报单）

**第1步**：打开报单链接
```
http://localhost:8080/loan-form.html
```

*局域网内可使用：http://192.168.1.16:8080/loan-form.html*

**第2步**：填写信息
- 客户姓名、电话、身份证
- 车型选择、车价、首付
- 贷款期数（12-60期）

**第3步**：点击提交
- 系统自动计算贷款金额
- 提交成功后显示订单号
- 完成！

### 风控员（审批报单）

**第1步**：下载飞书App
- App Store搜索"飞书"

**第2步**：加入风控群
- 联系管理员拉群

**第3步**：等待通知
- 新报单会在群里提醒
- 点击查看详情，审批决策

---

## 📝 填写示例

**必填字段**：
```
客户姓名：张三
联系电话：13800138000（11位）
身份证号：610XXXXXXXXXXXXXXX（18位）
业务员姓名：李四（自己的名字）
车型选择：奥迪A4（下拉选择）
车价：25（万元）
首付：5（万元）
贷款期数：36（3年）
```

**自动计算**：
```
贷款金额 = 车价 - 首付
例如：25万 - 5万 = 20万
```

## 🔍 查看进度

打开"我的报单"：
```
http://localhost:8080/my-orders.html
```

*局域网内可使用：http://你的Mac的IP:8080/my-orders.html*

**状态说明**：
```
🟡 待审批 - 等待风控员审核
🟢 已通过 - 可以办理贷款
🔴 已拒绝 - 审批未通过
```

页面每30秒自动刷新，实时更新状态！

## ❓ 常见问题

**Q：需要下载App吗？**
A：不需要！浏览器直接打开链接就能用。

**Q：手机能提交吗？**
A：可以！手机、电脑都能用。

**Q：提交后能修改吗？**
A：暂时不能。请仔细检查后再提交。

**Q：审批要多久？**
A：通常1-2个工作日。在"我的报单"里可以实时查看。

**Q：能看到别人的报单吗？**
A：不能。只能看到自己提交的报单。

**Q：忘记密码怎么办？**
A：本系统不需要登录，不需要密码。
