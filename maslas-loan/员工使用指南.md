# 🚗 汽车分期报单系统 - 员工使用指南

## 📱 功能简介

### 这是做什么的？
这是一个**汽车分期业务报单系统**，可以在线提交贷款申请、查看审批进度、接收通知提醒。

### 核心功能

| 功能 | 说明 |
|------|------|
| 📝 在线报单 | 手机/电脑上填写报单信息，快速提交 |
| 📋 进度查询 | 随时查看报单的审批状态 |
| 🔔 实时通知 | 新报单提醒、审批结果通知 |
| 💾 自动保存 | 提交的数据自动保存，不会丢失 |

## 📝 业务员使用指南

### 第一步：提交报单

1. **打开报单表单**
   - 点击链接：`http://your-domain.com/loan-form.html`

2. **填写客户信息**
   ```
   客户姓名：张三
   联系电话：13800138000（11位手机号）
   身份证号：610XXXXXXXXXXXXXXX（18位）
   ```

3. **填写车辆信息**
   ```
   业务员姓名：李四（自己的名字）
   车型选择：下拉选择（例如：奥迪A4、宝马3系）
   车价：25（万元）
   首付：5（万元）
   贷款期数：36期（3年）
   ```

4. **自动计算**
   - 系统会自动计算：贷款金额 = 车价 - 首付
   - 例如：25万 - 5万 = 20万

5. **备注说明（可选）**
   - 可以填写客户特殊情况或其他备注

6. **点击"提交报单"**
   - 系统会自动验证信息
   - 提交成功后会显示订单号

---

### 第二步：查看进度

1. **打开"我的报单"**
   - 点击链接：`http://your-domain.com/my-orders.html`

2. **查看报单列表**
   - 显示你提交的所有报单
   - 包含订单号、客户信息、车型、金额等

3. **查看审批状态**
   ```
   🟡 待审批 - 等待风控员审核
   🟢 已通过 - 审批通过，可以办理贷款
   🔴 已拒绝 - 审批未通过
   ```

4. **自动刷新**
   - 页面每30秒自动刷新一次
   - 状态更新后会自动显示

---

## 👨‍💼 风控员使用指南

### 第一步：接收通知

1. **加入风控审批群**
   - 下载飞书App
   - 加入"风控审批群"

2. **接收新报单提醒**
   - 有新报单时，群里会收到通知
   - 点击通知可以查看详情

### 第二步：审批报单

1. **在群里讨论**
   - 可以@其他风控员讨论
   - 查看客户信息和风险情况

2. **审批决策**
   - 通过：客户符合条件，可以办理贷款
   - 拒绝：客户不符合条件，说明拒绝原因

3. **通知业务员**
   - 审批结果会自动通知业务员
   - 业务员可以在"我的报单"里查看

✅ 确认贷款金额计算正确

## ❓ 常见问题

### Q1：提交报单后能修改吗？
**A**：暂时不支持修改。如需修改，请联系风控员。

### Q2：审批需要多长时间？
**A**：通常1-2个工作日，具体看情况。可以在"我的报单"里实时查看状态。

### Q3：手机能使用吗？
**A**：可以！手机、平板、电脑都能访问。

### Q4：忘记密码怎么办？
**A**：本系统暂时没有登录功能，不需要密码。

### Q5：报单提交失败怎么办？
**A**：检查网络连接，确认所有必填项已填写，再重新提交。

### Q6：能看到别人的报单吗？
**A**：不能。每个业务员只能看到自己提交的报单。

### Q7：审批结果会通知吗？
**A**：会。审批通过或拒绝后，你会在"我的报单"里看到状态更新。如果用飞书App，还会收到推送通知。

### Q8：可以导出报单数据吗？
**A**：暂不支持。需要的话可以联系管理员。

### Q9：系统会在什么时候更新？
**A**：如果有新功能或修复问题，会提前通知。

### Q10：遇到问题找谁？
**A**：
- 技术问题：联系管理员
- 业务问题：联系主管
